the original deck that the SlideController plugin was authored for.
To use it:

1. Copy your deck's SVGs and videos into `assets/` (or whatever path
   you prefer) next to this file.
2. In `deck.js`, update every `../../images/` path to point at your
   assets folder (e.g. `assets/`).
3. Add real markdown content for each slide below.
